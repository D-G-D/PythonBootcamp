## Modules

```
def has_zero(x, y, z):
    if x == 0 or y == 0 or z == 0:
        return True
    else:
        return False
```

```
In [9]: import myfuncs
```

```
In [10]: myfuncs.has_zero(1, -2, 0)
Out[10]: True
```

```
In [11]: from myfuncs import has_zero
```

```
In [12]: has_zero(1, -2, 3)
Out[12]: False
```
